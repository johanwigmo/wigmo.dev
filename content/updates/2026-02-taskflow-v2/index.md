TaskFlow v2.0 is now available on the App Store. This major update brings several new features and improvements.

## What's New

- Redesigned home screen with quick actions
- New habit tracking module
- Improved iCloud sync reliability
- Performance improvements across the board

## Bug Fixes

- Fixed an issue where tasks could be duplicated during sync
- Resolved a crash when editing recurring tasks
