TaskFlow is a productivity app designed for managing daily tasks and routines. Built with Swift and SwiftUI, it syncs seamlessly across devices using CloudKit.

## Features

- Daily task management with smart scheduling
- Habit tracking and streaks
- iCloud sync across all Apple devices
- Widgets for quick access

## Tech Stack

- **Language:** Swift
- **UI:** SwiftUI
- **Backend:** CloudKit
- **Platforms:** iOS, iPadOS, macOS
